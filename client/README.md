# Client

Apps in Toss WebView 기반 프론트엔드 작업 공간입니다.

## 현재 범위

- React + TypeScript + Vite
- `@apps-in-toss/web-framework` 기반 실행 환경
- 지역/카테고리/영업상태 중심의 초기 홈 화면 목업

## 실행

```bash
npm install
npm run dev
```

## 현재 화면 의도

- 앱 실행 직후 바로 서비스 목적이 보이는 런치 홈
- 지역 탐색 진입점 제공
- 카테고리 빠른 선택
- 최근 갱신 및 영업 상태 강조
- 향후 제보 흐름 확장 고려
