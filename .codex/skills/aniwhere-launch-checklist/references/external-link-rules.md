# External Link And App-Install Rules

Use this whenever Aniwhere adds links, share URLs, legal pages, partner pages, payment paths, or outbound CTAs.

## Forbidden

- Copy that directly asks the user to install an app.
- Images, banners, or badges that guide app installation.
- App Store, Play Store, or marketplace links.
- Install-benefit messaging such as "install to get a reward".
- External payment windows for miniapp commerce.
- Registered app functions that only work by leaving the miniapp.
- Share links that land on Aniwhere's own external site instead of the Toss share-link flow.

## Usually Allowed

- Legal notices and policy pages.
- Official public institution pages.
- Partner or venue information pages when they are informational only.
- Simple external information that is not a core registered app function.

## Review Heuristic

If the link is part of the core user journey, make it work inside the miniapp. If it is legal/reference information, keep the link narrow, clearly labeled, and free of install/payment inducement.
