## Key Official Criteria Captured

- Miniapp opens normally and app functions work inside the miniapp.
- Non-game Apps in Toss navigation bar is used.
- Back, home, close, more menu, and floating tab bar behavior is clear.
- User identity and required data persist after reopening.
- Light mode is the current design baseline.
- Permissions require prior user context and denial fallback.
- Bottom sheets must not auto-open or force action at inappropriate moments.
- CTAs must make the next action predictable.
- Ads must be predictable, preloaded, and reward only after completion.
