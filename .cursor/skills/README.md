# Cursor Skills Index

Team-discoverable skills live under `.cursor/skills`.

## Skill Discovery Check

Use this quick check when adding or changing skills:

1. Skill exists under `.cursor/skills/<skill-name>/SKILL.md`.
2. `name` and `description` exist in YAML frontmatter.
3. Skill name is unique across `.cursor/skills`.
4. Trigger terms in description match real user wording.
5. Related MCP docs are linked from `.cursor/mcp/*/docs`.
